# 프로젝트 개요
- Next.js(App Router, TypeScript) 웹앱
- 현재 기능: 뽀모도로 타이머 (25분 카운트다운)

# 명령어
- 개발 서버: npm run dev
- 빌드: npm run build

# 코딩 규칙
- 컴포넌트는 함수형 + TypeScript 타입 명시
- 상태는 React useState/useEffect 사용, 불필요한 라이브러리 추가 금지
- IMPORTANT: 비밀키·토큰은 코드에 하드코딩 금지 (.env 사용)

# 작업 방식
- 변경 전 플랜 모드로 계획 먼저 (3차시 워크플로)

# 커밋 규칙
- 메시지는 한국어로 작성
- 형식: 한 줄 요약 + 변경 이유
